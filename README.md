## Enterprise Network with OSPF Dynamic Routing

### Project Overview

Designed and implemented an enterprise network in Cisco Packet Tracer using **OSPF Dynamic Routing** to enable automatic route exchange between multiple routers. The network includes separate HR and IT departments connected through a core router, with DHCP for automatic IP assignment and VLANs for network segmentation.

### Technologies Used

* Cisco Packet Tracer
* OSPF Dynamic Routing
* DHCP
* VLANs
* IPv4 Addressing
* Layer 2 Switching

### Key Features

* Configured OSPF for dynamic route learning.
* Implemented DHCP for automatic IP allocation.
* Created VLANs for departmental separation.
* Established end-to-end connectivity between all networks.
* Verified routing tables, OSPF neighbors, and network communication.

### Learning Outcomes

* Enterprise Network Design
* OSPF Configuration and Troubleshooting
* VLAN Implementation
* DHCP Configuration
* Router and Switch Management

### Result

Successfully built a scalable enterprise network where all devices communicate through dynamically learned OSPF routes, simulating a real-world corporate network environment.
